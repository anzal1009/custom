Apply TDS Deducation for Vendor Bills and Payment Odoo Apps

Apply TDS Deduction on Vendor Payment Odoo App is used to apply/deduct TDS amount in vendor bill payment. TDS deduction is the process on Indian Accounting localization for vendor bill payment which is satisfied using our this app. This apps provide apply TDS checkbox including two fields 1) TDS account (visible accounts which only ticked as "TDS Account" Checkbox) and 2) TDS %. User can select TDS account and entered percentage of TDS then TDS amount is automatically calculated in total. User can also view TDS amount in vendor bill report which is deducted for vendor bill and register payment will auto deduct this TDS amount while reconcile vendor payment.

Features

Create Vendor Bill with TDS Amount

Users are allowed to create vendor bill with TDS amount and also select different type of TDS account.
